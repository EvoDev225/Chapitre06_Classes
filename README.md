# Jeu de devinette

Jeu en Java où il faut deviner un nombre aléatoire entre 1 et 100.

## Fonctionnement

- Un nombre aléatoire entre 1 et 100 est généré automatiquement
- Le joueur a **10 essais** pour trouver le bon nombre
- Après chaque essai, le jeu indique si le nombre cherché est plus grand ou plus petit
- Si le joueur trouve → **"Vous avez gagné !"**
- Si les 10 essais sont épuisés → **"Vous avez perdu !"**

## Exemple de partie

```
Saisissez un nombre :
50
C'est plus !
75
C'est moins !
63
Vous avez gagné ! Le nombre aléatoire est bien : 63
```
